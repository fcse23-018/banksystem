Detailed Screen-by-Screen Design
1. Login Screen (First screen)

Big PulaBank logo
Fields: Username / Customer ID + Password
Radio buttons: Login as → Customer | Admin
Login button
(Optional: "Forgot password")

2. Customer Dashboard (After customer login)
Sidebar Menu (Left):

Dashboard (shows current balance big in center)
Deposit
Withdraw
Transfer Money
Check Balance
Transaction History
Logout (at bottom)

Top Bar:

Welcome, [Customer Name]
Current Balance: P25,450.00
Logout button (red)

Main Pages:

Deposit → Enter amount → Confirm with PIN → Success message
Withdraw → Enter amount → Confirm PIN → (Check sufficient balance)
Transfer → Enter recipient Account Number + Amount + PIN
Transaction History → JTable showing Date, Type, Amount, Balance After

3. Admin Dashboard (After admin login)
Sidebar Menu (Left):

Dashboard (total customers, total money in bank)
Manage Customers
Add New Customer
Transaction Reports
Logout

Main Pages for Admin:

Manage Customers → Searchable JTable with columns:
| ID | Full Name | Account No | Phone | Balance | Actions (Edit/Delete/View) |
Add New Customer → Form with:
Full Name, Phone, ID Number, Initial Deposit, Password/PIN

Reports → Filter transactions by date/customer
